Public method respond

Constructs a response

Parameters:

  • req must be an Object.
    (HTTP(S) request Object)

  • res must be an Object.
    (HTTP(S) response Object)

  • output can be of any type.
    ([Optional] Response body)

  • status must be a Number.
    ([Optional] HTTP status code, default is 200)

  • headers must be an Object.
    ([Optional] HTTP headers to decorate the response with)

  • timer must be an Object.
    ([Optional] Date instance)

  • compress must be a Boolean.
    ([Optional] Enable compression of the response (if supported))

Returns an Objet
