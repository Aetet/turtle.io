Public method error

Error handler for requests

Parameters:

  • req must be an Object.
    (HTTP(S) request Object)

  • res must be an Object.
    (HTTP(S) response Object)

  • timer must be an Object.
    (Date instance)

Returns an Object
(Instance)

factory.prototype.error = function ( req, res, e, timer ) { e = e.message || e; timer = timer || new Date(); $.route.load( "error", req, res ); if ( this.config.probes ) { dtp.fire( "error", function () { return [req.headers.host, req.url, codes.SERVER_ERROR, e || messages.SERVER_ERROR, diff( timer )]; }); } };
