Public method cached

Verifies there's a cached version of the compressed file

Parameters:

  • filename must be a String.
    (Filename (etag))

  • format must be a String.
    (Type of compression (gzip or deflate))

  • fn must be a Function.
    (Callback function)

Returns an Objet
(Instance)

factory.prototype.cached = function ( filename, format, fn ) { var ext = REGEX_DEF.test( format ) ? ".df" : ".gz", path = this.config.tmp + "/" + filename + ext; fs.exists( path, function ( exists ) { fn( exists, path ); }); return this; };
