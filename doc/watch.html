src/watch.js

Public method watch

Watches path for changes & updated LRU

Parameters:

  • url must be a String.
    (LRUItem url)

  • path must be a String.
    (File path)

  • mimetype must be a String.
    (Mimetype of URL)

Returns an Object
(Instance)

factory.prototype.watch = function ( url, path, mimetype ) { this.sendMessage( MSG_REG_WAT, {url: url, path: path, mimetype: mimetype}, false, false ); };
